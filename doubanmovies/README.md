# 豆瓣即将上映信息爬取程序 #

----------

## 程序描述 ##
	
	**douban_01.py**
	1. 功能：爬取了豆瓣网即将上映电影的名称、时间和电影类型
	2. 实现要点：requests,BeautifulSoup(bs4)、lxml
	
	**douban_02.py**
	1. 功能：爬取了豆瓣网即将上映电影的名称、评分、导演和主演并打印json数据
	2. 实现要点：urllib2、HTMLParser

	**douban_03.py**
	1. 功能：爬取了豆瓣网即将上映电影的名称、时间、类型和国别
	2. 实现要点：requests,lxml(xpath)
